Requirements and Installation
=============================

Requirements
------------

Software
^^^^^^^^

- Python 2.7.10+

Python Modules
^^^^^^^^^^^^^^

- mg-tool-api
- pylint
- pytest

Installation
------------
Directly from GitHub:

.. code-block:: none
   :linenos:

   git clone https://github.com/Multiscale-Genomics/mg-process-test.git

Using pip:

.. code-block:: none
   :linenos:

   pip install git+https://github.com/Multiscale-Genomics/mg-process-test.git
